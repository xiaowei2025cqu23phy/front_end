# 第一部分 HTML - 网页的骨架

## 第 1 章 认识 Web 开发

### 1.1 Web 是如何工作的？

#### 生活中的比喻：餐厅点餐 🍽️

想象你去餐厅吃饭：

```
顾客 (你)          服务员         厨房
  │                  │              │
  │── 点菜 ─────────>│              │
  │                  │── 下单 ─────>│
  │                  │              │ 准备菜品
  │                  │<── 上菜 ─────│
  │<─ 端菜 ──────────│              │
  │                  │              │
```

**对应到 Web**：
```
浏览器 (用户)      HTTP 协议       服务器
  │                  │              │
  │── 请求 ─────────>│              │
  │                  │── 转发 ─────>│
  │                  │              │ 处理请求
  │                  │<── 响应 ─────│
  │<─ 显示网页 ──────│              │
  │                  │              │
```

#### 一次完整的网页访问

**步骤分解**：

```
第 1 步：你在浏览器输入网址
        www.baidu.com
        
        ↓
        
第 2 步：浏览器向百度服务器发送请求
        [HTTP 请求]
        "你好，我想访问百度首页"
        
        ↓
        
第 3 步：百度服务器收到请求
        处理请求，准备网页文件
        
        ↓
        
第 4 步：服务器返回响应
        [HTTP 响应]
        这是 HTML 文件、CSS、JavaScript...
        
        ↓
        
第 5 步：浏览器接收数据
        开始解析 HTML
        
        ↓
        
第 6 步：构建 DOM 树
        <html>
          <head>...</head>
          <body>
            <div>...</div>
          </body>
        </html>
        
        ↓
        
第 7 步：下载 CSS 和图片
        根据 HTML 中的链接下载资源
        
        ↓
        
第 8 步：渲染页面
        将内容绘制到屏幕上
        
        ↓
        
第 9 步：你看到了完整的网页
        ✨ 百度首页显示出来了！
```

**时间线**（以访问百度为例）：

50ms   ████████████░░░░░░░  DNS 解析

```

### 1.2 前端三剑客的关系

#### 形象比喻：建造房子 🏠

```
HTML  → 房子的框架结构
        （墙壁、房间、门窗的位置）

CSS   → 房子的装修装饰
        （油漆颜色、地板材料、灯具样式）

JS    → 房子的智能系统
        （电灯开关、自动门、安防系统）
```

#### 实际例子：制作一个按钮

**第一步：HTML 创建结构**
```html
<button>点击我</button>
```
这时候按钮长这样（浏览器默认样式）：
```
┌─────────────┐
│ 点击我       │ ← 灰色背景，黑色文字
└─────────────┘
```

**第二步：CSS 美化外观**
```css
button {
    background-color: #007bff;  /* 蓝色背景 */
    color: white;                /* 白色文字 */
    padding: 10px 20px;          /* 内边距 */
    border: none;                /* 无边框 */
    border-radius: 5px;          /* 圆角 */
    font-size: 16px;             /* 字体大小 */
}
```
现在按钮变成：
```
┌──────────────────┐
│   点击我          │ ← 蓝色背景，白色文字，圆角
└──────────────────┘
```

**第三步：JavaScript 添加交互**
```javascript
const button = document.querySelector('button');
button.addEventListener('click', function() {
    alert('你点击了按钮！');
});
```
现在点击按钮会弹出提示框：
```
┌──────────────────────┐
│  你点击了按钮！       │
│         [确定]        │
└──────────────────────┘
```

#### 三者关系图

```
         网页 (Web Page)
              │
    ┌─────────┼─────────┐
    │         │         │
    ▼         ▼         ▼
┌────────┐ ┌────────┐ ┌──────────┐
│  HTML  │ │  CSS   │ │JavaScript│
│ 结构层  │ │ 表现层  │ │  行为层   │
└────────┘ └────────┘ └──────────┘
    │         │         │
    │         │         │
    └─────────┴─────────┘
              │
              ▼
        完整的网页效果
```

**如果没有 CSS**：
```
网页会很丑：
白色背景
黑色文字
蓝色链接
没有任何样式
```

**如果没有 JavaScript**：
```
网页不能动：
点击没反应
无法验证表单
不能动态更新内容
```

**如果没有 HTML**：
```
根本没有网页！
```

#### 代码逐行解析

让我们一行一行地理解这段代码：

```html
<!DOCTYPE html>
```

**这是什么？**
- 这叫"文档类型声明"
- 告诉浏览器："我用的是 HTML5 版本"
- 不是 HTML 标签，所以没有结束标签

**为什么需要？**
```
想象你跟别人说话：
"嘿，我说的是中文！"

DOCTYPE 就是告诉浏览器：
"嘿，我用的是 HTML5！"
```

**位置要求**：
```
✅ 正确：必须在第一行
❌ 错误：前面有空行或空格
```

---

```html
<html lang="zh-CN">
```

**解释**：
- `<html>` 是 HTML 文档的根元素
- 所有其他标签都在它里面
- `lang="zh-CN"` 表示语言是简体中文

**好处**：
```
✓ 帮助搜索引擎知道网页语言
✓ 帮助屏幕阅读器选择正确的发音
✓ 帮助浏览器选择正确的字体
```

**结束标签**：
```html
</html>
```
注意最后一定要有这个，表示 HTML 文档结束。

---

```html
<head>
    <meta charset="UTF-8">
    <title>我的第一个网页</title>
</head>
```

**head 是什么？**
- 头部区域，存放"元数据"
- 这里的內容不会显示在页面上
- 是给浏览器和搜索引擎看的

**meta charset="UTF-8"**：
```
作用：设置字符编码为 UTF-8

为什么要 UTF-8？
因为 UTF-8 支持：
✓ 中文汉字
✓ 日文假名
✓ 韩文谚文
✓ 特殊符号（© ® € £ ¥）
✓ Emoji 表情（😀 😊 🎉）

如果不设置会怎样？
→ 可能显示乱码！
```

**title 标签**：
```
作用：设置网页标题

显示在哪里？
1. 浏览器标签页上
2. 搜索引擎结果中
3. 收藏夹/书签中

示例：
<title>我的第一个网页</title>
          ↓
    浏览器标签页显示："我的第一个网页"
```

---

```html
<body>
    <h1>你好，世界！</h1>
    <p>这是我的第一个网页。</p>
    <p>我正在学习前端开发！</p>
</body>
```

**body 是什么？**
- 主体区域
- 所有你能看到的内容都在这里
- 文字、图片、链接、按钮...

**h1 标签**：
```
<h1> → Heading 1（一级标题）

特点：
✓ 页面最重要的标题
✓ 字体最大
✓ 一个页面应该只有一个 h1

想象成一本书：
h1 = 书名
h2 = 章节名
h3 = 小节名
```

**p 标签**：
```
<p> → Paragraph（段落）

用于普通的文本段落
会自动在上下方添加一些空白
```

```
1. 文件没有保存

4. 代码有语法错误
   解决：检查标签是否正确闭合
```
